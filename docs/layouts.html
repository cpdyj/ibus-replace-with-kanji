  • かな文字は、50音の行ごとにまとまっています。テンテン(゛)をつけられるか・さ・た・は行、それから、や行のかな文字は左手がわにあります。
  • テンテン(゛)のついた字は、あとからテンテン(゛)のキー゛をおして入力します。

      たとえば: か → か、 か゛ → が

  • それぞれのキーの右がわにかかれている文字を入力するときは、前置・連続シフトキーをさきにおしてから、入力したい文字のキーをおします。

      たとえば: 前置・連続シフト あ → あ

  • 「ぱぴぷぺぽ」のマル(゜)は、あとからマル(゜)のキー゜をおして入力します。

      たとえば: は → は、 は前置・連続シフト ゜ → ぱ

  • 前置・連続シフトキーをおしたままにしていても、それぞれのキーの右がわにかかれている文字を入力できます。

      たとえば: 前置・連続シフトをおしたまま、ひ ゜ → ぴ

  • ちいさな「っ」はっとかかれているキーをおして入力します。
  • ちいさな「ゃゅょぁぃぅぇぉ」は、あとからテンテン(゛)のキー゛をおして入力します。

      たとえば: よ → よ、 よ゛ → ょ
